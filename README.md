no installation necessary just visit the deployed app site 

https://expressjsnotetakermc11-app.herokuapp.com/

## Usage

simply type the title and text of a note you want to save then click the save button (disk icon)   on the top right. A list of your current saved notes will appear on your left, and a trash can icon will appear next to them if you want to delete them.
